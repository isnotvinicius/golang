# Estudos de Golang

Este repositório tem como objetivo salvar todos os meus estudos com a linguagem Golang. Ele estará separado em pastas, cada pasta com um projeto e temas diferentes relacionados a linguagem.

Todos os projetos foram feitos seguindo os cursos de Golang da `Alura`, portanto todo o conteúdo do código e explicações são de crédito deles. 

Cada pasta terá um `README.md` explicativo. Gostaria de salientar que estas explicações são a minha visão e entendimento do que foi passado durante o curso.
